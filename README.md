jQuery Circular Text Plugin
===========================

A plugin to render text along a circular path, creating a fancy button effect.

## How To Use

JS:
```js
jQuery('.circular-text').circularText({
  speed: 2
});
```

# Reference
**Options :**
  
  * `speed` : The rotational speed when using animating.
